Hello! Thank you for checking out my Minesweeper game! :D

First of all, the purpose of the game is to reveal all the cells in the grid excluding the cells containing mines. The first cell revealed will never be a mine.

After revealing the first cell, look out for the number written in each one. the number corresponds to the number of mines in the adjacent cells. Clicking a mine cell will cause you to lose the game :(
Keep revealing cells without mines until they are all revealed to win the game!

Current features of this game include:
1. File menu allows the player to start a new game at any time.
2. Winning or losing the game causes the grid buttons to disable.
3. Right clicking a blue cell causes it to turn yellow, which flags it as a potential mine, decreasing the mine counter and preventing accidental clicking. Right clicking again disables the flagged status.
4. The mine counter at the bottom right of the screen can help track the number of mines left unmarked.
5. Empty cells clear out all the adjacent empty cells.

Potential future additions to the game:
1. Timer to calculate the time spent on the game
2. Statistics and scores of best times
3. More difficulty and customisable grid height, width, and mine count
4. Better visuals and sounds
